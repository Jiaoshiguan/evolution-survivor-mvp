# Evolution Survivor MVP

这是一个用 `Godot 4.5.x` 直接落地的短局刷潮原型，目标不是完整产品，而是验证下面这条闭环：

- 单人俯视角移动
- 自动攻击和刷潮
- 敌群压力随时间升级
- 升级时随机三选一
- 局外 `DNA` 永久成长

## 为什么这一版选 Godot

- 对 AI 最友好：场景和脚本都是纯文本，便于直接生成和修改
- 对原型最快：不依赖重型编辑器元数据，适合在空目录里直接起工程
- 对独立商业也可行：MIT 协议，没有 Dota 2 创意工坊那种平台绑定问题

## 当前 MVP 内容

- 主菜单
- 1 个英雄核心
- 5 类敌人（含精英与终局体）
- 9 个随机强化
- 3 个局外永久升级
- 8 分钟单局计时

## 运行方式

1. 安装或解压 `Godot 4.6.x` Windows 标准版。
2. 打开本目录中的 `project.godot`。
3. 运行主场景。
